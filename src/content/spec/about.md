# About
Isandrel’s Blog is a personal technical blog about software, tools, and systems.

Posts in the **Examples** category are retained demonstrations of features from the [Fuwari](https://github.com/isandrel/fuwari) theme. They remain alongside original technical articles so the theme’s supported Markdown and presentation features can be explored on the live site.
